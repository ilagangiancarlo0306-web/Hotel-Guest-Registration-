OVERVIEW

Hotel Guest Registration System is a simple web-based application developed using HTML, CSS, and JavaScript designed to register and manage hotel guest information efficiently.
The system allows users to input guest details such as first name, last name, email, phone number, address, and date of birth through a user-friendly registration form. Submitted 
data is dynamically displayed in a registered guests table, demonstrating basic CRUD concepts and client-side interactivity.

FEATURES

-Guest registration form with fields for personal information

-Client-side form validation using JavaScript

-Dynamic display of registered guests in a table

-Clean and user-friendly interface

-Responsive layout for better usability on different screen sizes

-Demonstrates basic CRUD concepts (Create and Read)
